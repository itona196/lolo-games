# Lolo Games

**Lolo Games** est un mini-jeu web développé en **HTML, CSS et JavaScript**.  
Le jeu fonctionne directement dans le navigateur, sans dépendance externe.

---

## Description

Ce projet est un jeu simple conçu pour être lancé localement via un navigateur web.  
Il repose uniquement sur des technologies web standards et ne nécessite aucune installation spécifique.
